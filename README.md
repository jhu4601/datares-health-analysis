# Data Over Diets: Rethinking Weight Loss
**DataRes at UCLA** | Data Journalism Team

## Overview
The conventional wisdom is simple: consume fewer calories than you burn. But the 
data tells a far more complex story. This project analyzes a comprehensive health 
dataset to identify which factors truly drive weight change, and the results are 
surprising.

## Key Findings
- **Stress dominates caloric intake.** High-stress individuals (stress score 10/10) 
  trend noticeably heavier than low-stress individuals even at identical calorie levels
- **Workout type doesn't matter — consistency does.** Strength training, yoga, and 
  cardio all produced nearly identical weight change distributions, suggesting that 
  finding a workout you can sustain matters more than which one you pick
- **Moderate intensity + longer duration wins.** 8–12 week regimens at moderate 
  intensity produced the most weight loss per week, outperforming short, intense bursts
- **Room temperature correlates with weight change** (r = 0.38), suggesting 
  environmental factors subtly influence metabolism and behavior
